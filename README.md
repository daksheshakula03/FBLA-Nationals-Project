## Game Title
2000s Trivia Challenge

## About the game
- This a single user multi question trivia game for 2000 decade.
- The game takes the player through historical fact journey in 2000s 

## Features
- Game can be played in Guest Mode, existing user or register as a new user.
- Three categories with multiple questions
- There is Dark Mode and Light Mode
- Start music can be muted
- There is a bonus round at the end of each category but if you want you could also skip the bonus.
- Each category at the end has VIEW Facts for the user to know more about that decade in that category.
- A score report is generated at the end of the game
